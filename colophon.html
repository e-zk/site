Colophon

This website is a collection of static HTML pages generated by a custom site generator. It is a VM on “someone else’s computer” running OpenBSD. The pages are served by relayd, and httpd. The entire content of the site can be found in a git repository.

A hidden service mirror of this site is available and can be accessed at the following onion address: 64wv2uqwjacqer7z5d6ooqgrvjwlioizmo7hgmxm7zxerbvgnoqhafid.onion.

Posts are edited in the vis editor as Markdown, or more recently, djot files. In the footer of most pages, you can open the plaintext representation. I try to keep the text width under 80 characters for consistency.

Design

The site’s design is inspired by many other personal sites, too many to remember. Inspiration has also been taken from any number of sites on Dead Simple Sites.

Code font is Commit Mono.

CSS is hand-written by yours truly.
